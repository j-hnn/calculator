Calculator
John Weng

Description:
	This is a simple calculator made using GoDot. It has all the simple functions. There are sounds for the different buttons. 
	
How to use:
	click the buttons corresponding to the actions you want to do
	
Buttons:
	c - clear
	± - change sign
	← - delete
	÷ - divide
	x - multiply
	- - subtract
	+ - add
	= - equals
